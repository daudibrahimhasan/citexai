# CiteXai - AI-Powered Citation Verification

**Stop Citing Fake Papers.** CiteXai is a robust API and web application designed to verify academic citations against multiple authoritative databases (CrossRef, OpenAlex, Google Books, arXiv) to detect hallucinations and validate research.

> **Built by [@daudibrahimhasan](https://github.com/daudibrahimhasan)**  
> **A product of Nexasity Ai**

---

## üöÄ Features

- **Multi-Database Verification:** Cross-checks citations against 400M+ papers and books.
- **Hallucination Detection:** Instantly flags fake or "dreamed" papers generated by LLMs.
- **AI Citation Repair:** Automatically fixes malformed citations and appends valid DOIs or arXiv IDs.
- **Hybrid Detection:** Identifies subtle errors (wrong year, fake journal) in otherwise real papers.
- **Frontend & API:** Includes a Next.js web interface and a REST API.

## üì¶ Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/daudibrahimhasan/citexai.git
    cd citexai
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Configure Environment:**
    Copy `.env.example` to `.env.local` and add your keys:

    ```bash
    cp .env.example .env.local
    ```

    _Required: `GROQ_API_KEY` for AI fix functionality._

4.  **Run Locally:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000).

## üîå API Usage

### Verify Citation

**POST** `/api/verify`

```json
{
  "citation": "Vaswani, A., et al. (2017). Attention is all you need."
}
```

**Response:**

```json
{
  "verified": true,
  "score": 100,
  "status": "verified",
  "message": "VERIFIED - Found in arXiv (100%)",
  "details": { ... }
}
```

### Fix Citation

**POST** `/api/fix-citation`

```json
{
  "citation": "Attention needs you all (2017) Vaswani"
}
```
